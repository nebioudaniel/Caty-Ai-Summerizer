## Introduction

Caty GPT-4 Article Summarizer is designed to help you quickly understand the key points of long articles. Using the power of OpenAI's GPT-4, Caty extracts and delivers concise summaries, making it easier for you to consume information efficiently.

## Features

- **Accurate Summaries**: Provides clear and concise summaries of lengthy articles.
- **Easy to Use**: Simple interface for pasting articles and generating summaries.
- **Open Source**: Contributions and improvements are welcome from the community.

## Installation

To install and run Caty GPT-4 Article Summarizer locally, follow these steps:

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/nebioudaniel/Caty-Ai-Summerizer.git
    cd Caty-Ai-Summerizer
    ```

2. **Install Dependencies**:
    ```sh
    npm install
    ```

3. **Set Up Environment Variables**:
    Create a `.env` file in the root directory and add your OpenAI API key:
    ```sh
    REACT_APP_OPENAI_API_KEY=your_openai_api_key
    ```

4. **Start the Development Server**:
    ```sh
    npm start
    ```

## Usage

1. Open your web browser and navigate to `http://localhost:3000`.
2. Paste the article you want to summarize into the input field.
3. Click the "Summarize" button to get a concise summary of the article.

You can configure additional settings in the `tailwind.config.js` file and other configuration files as needed.
